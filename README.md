# 🎯 Landing Page Paintball

## 📌 Descripción del Proyecto

Este proyecto consiste en una landing page diseñada para promocionar servicios y experiencias de paintball. La página fue desarrollada con el objetivo de ofrecer una interfaz visual atractiva, dinámica y fácil de navegar para los usuarios interesados en actividades recreativas y deportivas.

La landing page presenta información sobre el servicio de paintball, imágenes llamativas, secciones informativas y un diseño responsive que permite una correcta visualización en diferentes dispositivos.

## 🚀 Características

* Diseño moderno y atractivo
* Interfaz responsive
* Navegación sencilla
* Secciones informativas sobre paintball
* Uso de HTML, CSS y JavaScript
* Despliegue mediante GitHub Pages

## 🛠️ Tecnologías Utilizadas

* HTML5
* CSS3
* JavaScript
* Google AI Studio/Gemini

El propósito de este proyecto es practicar el desarrollo de interfaces web tipo landing page, aplicando conceptos de diseño web, estructura semántica, estilos visuales y adaptación responsive.
